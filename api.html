API Documentation

Pharmacophore

openpharmacophore.Pharmacophore([elements])

Native object for pharmacophores.

openpharmacophore.Pharmacophore.add_element(...)

Adding a new element to the pharmacophore.

openpharmacophore.Pharmacophore.add_to_NGLView(view)

Adding the pharmacophore representation to a view (NGLWidget) from NGLView.

openpharmacophore.Pharmacophore.show([palette])

Showing the pharmacophore model together with the molecular system from with it was extracted as a new view (NGLWidget) from NGLView.

openpharmacophore.Pharmacophore.to_ligandscout(...)

Method to export the pharmacophore to the ligandscout compatible format.

openpharmacophore.Pharmacophore.to_pharmer(...)

Method to export the pharmacophore to the pharmer compatible format.
