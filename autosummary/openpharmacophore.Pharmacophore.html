openpharmacophore.Pharmacophore

class openpharmacophore.Pharmacophore(elements=[])[source]

Native object for pharmacophores.

Parent class of LigandBasedPharmacophore, StrucutredBasedPharmacophore and Dynophore Openpharmacophore native class to store pharmacophoric models. A pharmacophore can be constructed from a list of elements or from a file.

Parameters

elements (list of openpharmacophore.pharmacophoric_point.PharmacophoricPoint) – List of pharmacophoric elements

Variables

  • elements (list of openpharmacophore.pharmacophoric_point.PharmacophoricPoint) – List of pharmacophoric elements

  • n_elements (int) – Number of pharmacophoric elements
